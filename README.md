# Year 10 Digital Technologies Weekly Guide App

A separate student instruction app for the Weather Emergency Alert App program.

## Purpose
Students use this app for weekly instructions and scaffold generation. They copy the generated scaffold into the Student Evidence App, then customise and complete their evidence there.

## Includes
- Semester 1 IA1 Weather Alert MVP weekly guide
- Semester 2 IA2 Final Build weekly guide
- Copy-ready weekly scaffolds
- Downloadable scaffold text files
- Teacher checkpoint guide

## Run locally
```bash
npm install
npm run dev
```

## Build for deployment
```bash
npm run build
```

Deploy the project folder to Vercel, Netlify, Render Static Site, or any Vite-compatible host.
